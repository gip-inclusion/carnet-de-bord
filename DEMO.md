# Comptes de démo

Il existe plusieurs roles dans l'application Carnet de bord. Chacun correspondant à un profil d'usager.
- admin_cdb
- admin_pdi (manager dans les permissions hasura)
- admin_structure
- orientation_manager
- professional
- beneficiary
